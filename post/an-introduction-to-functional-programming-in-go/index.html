Functional Programming with Go Generics

November 22, 2021

One of my favorite programming books is “An Introduction to Functional Programming” by Richard Bird & Philip Wadler. In it, the authors show how a rich set of programming functions can be built upon simple constructs like lists, pattern matching and recursion. This week I was reading the Go Generics proposal and it got me thinking - could we build the same set of functions using Go Generics?
